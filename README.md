#  Farmaora
### AI-Powered Agricultural Investment Platform
> Built for Keheilan Asset Management AI Hackathon

---

## What is Farmaora?
Farmaora makes agricultural investing accessible to everyone — including young investors with limited budgets — by combining smart algorithms and AI to match investors with the right farms worldwide.

---

## AI Features Built

| Feature | Type | Description |
|---------|------|-------------|
| Dynamic Risk Scoring | Algorithm | Calculates farm risk based on sustainability score + investor budget |
| Investor-Farm Matching | Algorithm | Ranks farms by compatibility with investor profile |
| Harvest Validation | Algorithm | Validates operator harvest reports against crop benchmarks |
| KYC Document Screening | Gemini AI | Extracts and verifies identity documents |
| Natural Language Search | Algorithm | Converts plain English queries into database filters |

---

## Tech Stack
- **Frontend:** React + Vite
- **Backend:** Node.js + Express
- **Database:** Supabase (PostgreSQL)
- **AI:** Google Gemini API
- **Auth:** Supabase Auth

---

## Investment Models Supported
1. **Fractional Land** — own a share of farmland (5-10 year horizon)
2. **Farm Operations** — fund the harvest cycle (1-3 year horizon)
3. **Hybrid** — combine land + operations (3-7 year horizon)
